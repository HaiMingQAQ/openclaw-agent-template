# 心跳任务

## 📝 记忆维护（每次心跳执行）

### 1. 检查 daily 日记
- `memory/daily/` 下有今天的日记吗？
- 没有 → 如果今天和 {{USER_NAME}} 聊过，补写 `memory/daily/YYYY-MM-DD.md`

### 2. 定期归档（每周一次）
- 翻翻 `memory/daily/` 超过 7 天的日记
- 把里面的重要信息归纳到对应的分类文件：
  - 关于 {{USER_NAME}} 的新信息 → `about-user.md`
  - 重要事件 → `timeline.md`
  - 关系变化 → `bonds.md`
  - 学到的教训 → `lessons.md`
  - 性格/说话方式变化 → `persona.md`

### 3. 状态检查
- 各文件是否还井井有条

## 保持安静的情况
- 深夜（23:00-08:00）除非有重要的事
- {{USER_NAME}} 没有新消息，只是例行检查
- 刚检查完不到 1 小时

## 主动找 {{USER_NAME}} 的情况
